## 📦 설치

<details>
<summary><b>📥 빠른 설치 가이드</b></summary>

```bash
# 1. 저장소 복제
git clone https://github.com/yourusername/ultratranslate.git

# 2. Chrome 확장 프로그램 페이지 열기
chrome://extensions/

# 3. 개발자 모드 활성화
토글 → 켜기

# 4. 확장 프로그램 로드
"압축해제된 확장 프로그램을 로드합니다" 클릭 → 폴더 선택
```

</details>

## ⚙️ 구성

### 🚀 빠른 설정
1. 브라우저 도구 모음에서 UltraTranslate 아이콘 클릭
2. 드롭다운에서 대상 언어 선택
3. 두 텍스트를 모두 유지하려면 "원문 보존" 토글
4. "페이지 번역"을 클릭하여 번역 시작

### 🔑 API 구성

<details>
<summary><b>Google 번역 (기본값)</b></summary>

- ✅ 구성 불필요
- ✅ 무료 사용
- ✅ 즉시 사용 가능

<details>
<summary><b>DeepSeek API</b></summary>

1. [DeepSeek 플랫폼](https://platform.deepseek.com/)에서 API 키 받기
2. 확장 프로그램 설정 열기 (팝업의 ⚙️ 버튼)
3. API 구성 섹션으로 이동
4. 드롭다운에서 "DeepSeek" 선택
5. API 키 입력
6. 설정 저장

</details>

<details>
<summary><b>OpenAI API</b></summary>

1. [OpenAI 플랫폼](https://platform.openai.com/)에서 API 키 받기
2. 확장 프로그램 설정 열기
3. 드롭다운에서 "OpenAI" 선택
4. API 키 입력
5. 선택사항: 번역 프롬프트 사용자 정의
6. 설정 저장

</details>

<details>
<summary><b>Baidu 번역 API</b></summary>

1. [Baidu 번역](https://fanyi-api.baidu.com/)에서 APP ID와 비밀 키 받기
2. 확장 프로그램 설정 열기
3. 드롭다운에서 "Baidu" 선택
4. 형식으로 자격 증명 입력: `appid:secretkey`
5. 설정 저장

</details>

## 🎮 사용법

### 📖 기본 번역
1. 번역하려는 웹페이지로 이동
2. UltraTranslate 아이콘 클릭
3. "페이지 번역" 클릭
4. 페이지가 번역될 때 진행률 표시기 확인
5. 원문은 투명도가 낮아져 표시되고, 번역은 녹색으로 강조 표시

### ⏹️ 번역 중지
- "번역 중지" 버튼을 클릭하여 진행 중인 번역 중단
- 큰 페이지나 취소하려는 경우에 유용

### ⌨️ 키보드 단축키
- `Alt+T`: 번역 켜기/끄기 토글
- `Alt+O`: 원문 가시성 토글

### 🤖 사용자 정의 번역 프롬프트 (AI API만 해당)
1. 설정 페이지 열기
2. 번역 섹션으로 이동
3. 텍스트 영역에 사용자 정의 프롬프트 입력
4. 대상 언어의 자리 표시자로 `{language}` 사용
5. 예: "{language}로 번역하되, 전문 용어는 유지하고 간결하고 자연스럽게"

## 🔧 고급 설정

### 💾 캐시 관리
- **캐시 크기**: 현재 캐시된 번역 수 보기
- **캐시 제한**: 최대 1000개 번역 저장
- **캐시 만료**: 기본 24시간
- **캐시 지우기**: 모든 캐시된 번역 제거

### 🎨 외관 사용자 정의
- **번역 스타일**: 강조 표시, 밑줄, 버블 또는 나란히 선택
- **번역 색상**: 강조 색상 사용자 정의
- **글꼴 크기 조정**: 번역된 텍스트 크기 조정 (50-150%)
- **원문 투명도**: 원문 가시성 조정 (0-100%)

## 🏗️ 기술 아키텍처

### 📋 구성 요소
- **Manifest V3**: 최신 Chrome 확장 프로그램 아키텍처
- **Content Script**: 웹페이지에 번역 기능 주입
- **Background Service Worker**: API 호출 및 메시지 라우팅 처리
- **Options Page**: 전체 설정 관리 인터페이스
- **Popup Interface**: 빠른 액세스 제어

### 🔄 텍스트 처리 파이프라인
1. **텍스트 감지**: 효율적인 DOM 순회를 위한 TreeWalker API와 NodeFilter
2. **필터링**: 번역 불가능한 콘텐츠를 건너뛰는 포괄적인 규칙
3. **세분화**: 정확한 문장 분할을 위한 Intl.Segmenter API
4. **배치 처리**: 컨텍스트와 유사성에 따른 지능적인 그룹화
5. **번역**: 병렬 API 호출
6. **표시**: 원문 보존과 함께 DOM 조작

### ⚡ 성능 최적화
- **LRU 캐시**: 중복 API 호출 감소
- **배치 처리**: 효율성을 위해 유사한 텍스트 그룹화
- **동시 번역**: 여러 배치 동시 처리
- **디바운스된 변경**: 과도한 재번역 방지
- **WeakMap 저장소**: 메모리 효율적인 원문 저장

## 🔒 개인정보 보호 및 보안

- ✅ 모든 번역은 공식 API 엔드포인트를 통해 처리
- ✅ API 키는 Chrome의 보안 저장소에 저장
- ✅ 제3자에게 데이터 수집 또는 전송 없음
- ✅ 원본 페이지 콘텐츠는 영구적으로 수정되지 않음
- ✅ 캐시는 로컬에 저장되고 자동으로 만료

## 🐛 문제 해결

### 설정 페이지가 열리지 않음
- 확장 프로그램 아이콘을 마우스 오른쪽 버튼으로 클릭하고 "옵션" 선택
- 또는 `chrome://extensions/` → UltraTranslate → 세부정보 → 확장 프로그램 옵션

### 번역이 작동하지 않음
1. API 키가 구성되었는지 확인 (Google 이외의 서비스)
2. 페이지 로딩이 완료되었는지 확인
3. 페이지 새로고침 시도
4. 콘솔 오류 확인 (F12 → 콘솔)
5. 사이트가 제외 목록에 없는지 확인

### 번역이 느림
- 설정에서 배치 크기 줄이기
- 캐시가 너무 크면 지우기
- 인터넷 연결 확인
- Google 번역 사용 고려 (가장 빠름)
